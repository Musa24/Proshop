## Proshop
ProShop
This Project was online shop for selling products and paying by using PayPal, this project is inspired by Amazon.
### The technologies used:
     •	mongo DB (mongoose)
     •	NodeJS (express)
     •	React JS
     •	Redux
     •	React-redux
     •	Redux-thunk
     •	PayPal
     •	Bootstrap
     •	Many More
